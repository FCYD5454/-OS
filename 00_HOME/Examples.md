# 💡 個人 OS：實作範例 (Examples)

這份文件展示了如何將碎片化的生活點滴，透過「個人 OS」轉化為系統化的自我認知。

---

## 範例 1：從 Daily 到 Pattern (行為模式的發現)

### 第一步：Daily 日誌 (40_LOGS/2026-03-12.md)
```md
---
type: daily
date: 2026-03-12
mood: 焦慮
energy: 3
tags: [log/daily]
themes: [Theme/被評價的恐懼]
---
## 今日一幕
下午開會時，主管問我對專案的看法，我腦袋一片空白，只說了「沒意見」，然後整場會議都在滑手機。

## 身體訊號
胸口很悶，手心出汗，呼吸變得很淺。

## 情緒翻譯
我感到恐懼。我其實在保護自己不被別人看穿「我其實不懂」。
```

### 第二步：建立 Pattern 卡片 (30_SYSTEMS/Pattern-會議沉默.md)
*當你發現這種情況發生了三次以上，就該建立模式卡了。*
```md
---
type: pattern
domain: work
tags: [system/pattern]
triggers: [被點名發言, 面對權威者]
payoff_short_term: 避免了當場出醜的風險
cost_long_term: 失去專業信用, 內在焦慮累積
---
## 觸發 → 反應鏈
情境：被點名 -> 想法：「我說錯怎麼辦？」 -> 行為：沉默/滑手機逃避。

## 這個模式在「幫我」什麼？
它在保護我那脆弱的「完美形象」。
```

## 範例 2：從 Memory 到 Rule (核心規則的挖掘)

### 建立 Memory 卡片 (20_LIFE_STORY/Memory-小學演講.md)
```md
---
type: memory
period: childhood
tags: [story/memory]
themes: [Theme/羞恥感, Theme/完美主義]
---
## 片段
小學三年級朗讀比賽，我唸錯了一個字，全班大笑，老師搖了搖頭。

## 當時的我在學什麼規則？
「只要犯錯，我就會被嘲笑，且失去權威者的愛。」

## 這規則今天仍適用嗎？
不適用。成年人的世界允許錯誤，且錯誤是學習的機會。
```

## 範例 3：建立 Person 卡片 (30_SYSTEMS/Person-主管.md)

```md
---
type: person
relationship: mentor
closeness: 4
tags: [system/person]
---
# 主管 A

## 🎭 互動模式
- **我的角色**: 討好者 (People Pleaser)。
- **固定劇本**: 他提出要求 -> 我明明沒時間卻說好 -> 我熬夜完成並感到憤怒。

## 💡 觀察與反思
他讓我看到了我對「權威」的過度恐懼。我投射了父親的形象在他身上。
```

## 範例 4：從價值觀衝突到決策 (Value Conflict)

### 建立 Value 卡片 (10_IDENTITY/Value-自由.md)
```md
---
type: value
tags: [identity/value]
importance: 10
---
# 價值觀：自由
- **願付代價**: 願意忍受收入的不穩定。
- **衝突時刻**: 當公司要求我簽署一份限制副業的合約時，我感到了極大的掙扎。
```

### 建立 Weekly 總結 (40_LOGS/Weekly-2026-W11.md)
```md
---
type: weekly
tags: [log/weekly]
---
## 本週發現
這週在「穩定收入」與「工作自由」之間掙扎。
我發現我的舊規則「沒錢就不安全」正在與新價值「自由」打架。
下週行動：列出維持基本生活的最低成本，降低對「沒錢」的非理性恐懼。
```

## 範例 5：身心狀態的連結 (Mind & Body)

### 在 Daily 中記錄身體訊號
```md
---
type: daily
themes: [Theme/界線]
---
## 身體訊號
今天下午跟某個朋友聊天後，感到胃部緊縮，像被揪住一樣。
## 情緒翻譯
我感到「被侵犯」。他一直在打聽我的私事，而我沒有拒絕。
這反映了我的 [[30_SYSTEMS/Relationship-Map|關係系統]] 中「難以說不」的模式。
```

## 🚀 如何閱讀你的系統？
當你寫完以上範例後，打開 `00_HOME/Index.md`：
1. **最近的模式** 表格會出現「會議沉默」。
2. **童年回憶** 表格會出現「小學演講」。
3. **Graph View** 會顯示 `[[Theme/被評價的恐懼]]` 連結了這兩件事。

**這就是歸納自己的過程：從「我今天很煩」變成「我發現我有個關於『被評價』的保護機制」。**
